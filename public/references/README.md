# public/references — фото-референсы материалов для AI-визуализации

Сюда кладём фотографии материалов (образцы фактур/профилей), которые Gemini
использует как референс при генерации фасада.

## Структура
```
references/
  foundations/   цоколь / фундамент
  frames/        обрамление окон
  columns/       угловые колонны
  belts/         межэтажные пояса
  brackets/      кронштейны
  termopanels/   термопанельные планки
  decor/         декор (наличники, пилястры, карнизы)
  textures/      текстуры стен (травертин и т.п.)
```

## Как добавить материал (2 шага)
1. Положи фото сюда: `references/<папка>/<id>.jpg` (формат **.jpg**).
2. Добавь строку в соответствующий каталог в `lib/` (напр. `lib/columns.ts`)
   с тем же `id`. Имя файла = `id`.

Если фото для `id` нет — в интерфейсе покажется цветной квадрат (swatch),
а Gemini получит текстовую подсказку (`hint`). Всё работает и без фото.

⚠️ ВАЖНО (защита от галлюцинаций): в один запрос к Gemini уходит не больше
**4 фото-референсов** (см. `MAX_REFERENCE_IMAGES` в `app/api/visualize/route.ts`).
Лишние выбранные материалы подставляются текстом, а не картинкой.
